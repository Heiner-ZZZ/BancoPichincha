# 🏦 Diseño Responsivo de Inicio de Sesión - Banco Pichincha

## 📌 Descripción del proyecto

Este proyecto corresponde al diseño de una interfaz móvil inspirada en la pantalla de inicio de sesión de Banco Pichincha. La actividad fue desarrollada en Android Studio utilizando XML para construir la interfaz gráfica mediante `ConstraintLayout`, `LinearLayout`, `TextView`, `ImageView` y componentes visuales tipo tarjeta.

El objetivo principal del proyecto es recrear una pantalla de acceso bancaria con una distribución visual ordenada, responsiva y adaptable a dispositivos móviles, manteniendo una apariencia limpia y cercana al diseño original de la aplicación.

## 🎯 Objetivo de la tarea

Diseñar una pantalla de inicio de sesión para una aplicación móvil, aplicando principios de diseño responsivo y organización de elementos visuales en Android Studio.

La interfaz incluye:

* 🏦 Encabezado con logo y nombre de la entidad.
* 🏷️ Texto de versión de la aplicación.
* 🖼️ Imagen principal representativa del inicio de sesión.
* 🔐 Mensaje de seguridad para el usuario.
* 👤 Opciones de acceso mediante usuario y contraseña, huella o Face ID, y PIN.
* 📍 Accesos rápidos a ubicación, clave digital y llamada telefónica.
* 👥 Enlace visual para ingresar con otro usuario.

## 🛠️ Tecnologías utilizadas

* 🤖 Android Studio
* 📄 XML
* 🧩 ConstraintLayout
* 📦 LinearLayout
* 🎴 MaterialCardView
* 🖼️ ImageView
* 🔤 TextView
* 🎨 Recursos gráficos en drawable

## 🧱 Estructura principal del diseño

El diseño se encuentra construido principalmente en el archivo XML de la actividad principal.

El layout utiliza un `ConstraintLayout` como contenedor general, permitiendo ubicar los elementos de forma flexible dentro de la pantalla. Además, se emplean `LinearLayout` horizontales para organizar las opciones de acceso y los accesos rápidos inferiores.

Los elementos visuales principales son:

* 🖼️ `ImageView` para mostrar el logo y la imagen central.
* 🔤 `TextView` para los textos informativos y botones visuales.
* 🎴 `MaterialCardView` para representar las tarjetas de métodos de ingreso.
* 🎨 Iconos personalizados ubicados en la carpeta `drawable`.

## 📁 Recursos utilizados

Para el desarrollo de la interfaz se utilizaron imágenes e iconos almacenados en la carpeta `drawable`, tales como:

* 🏦 Logo de Banco Pichincha.
* 🖼️ Imagen principal de inicio de sesión.
* 👤 Icono de usuario.
* 🔐 Icono de huella digital.
* ➕ Icono de más opciones.
* 📍 Icono de ubicación.
* 🗝️ Icono de clave.
* 📞 Icono de llamada.

## ✅ Resultado final

El resultado final es una pantalla móvil visualmente organizada, con estructura responsiva y elementos distribuidos de forma similar a una aplicación bancaria real. La interfaz permite visualizar diferentes métodos de acceso y opciones rápidas para el usuario, manteniendo una identidad visual basada en colores corporativos y componentes modernos de Android.
